# OrionVim

Custom NvChad configuration aimed to semplicity and ease of use, while still maintaining its core features.

Command to install all LSP's

```bash

:MasonInstall codelldb prisma-language-server clangd pyright vtsls selene lua-language-server css-lsp eslint-lsp html-lsp json-lsp shellcheck shfmt solang stylua tailwindcss-language-server yaml-language-server

```
